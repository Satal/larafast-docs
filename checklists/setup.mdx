---

- [ ] Install `composer i && npm i && cp .env.example .env && php artisan key:generate && php artisan migrate:fresh --seed`
- [ ] Update your `.env` specifying things like:
  -  [ ] `APP_NAME`
  -  [ ] mail settings
  -  [ ] Stripe keys
  -  [ ] Cashier Currency (if not USD)
- [ ] Replace `/public/images/logo.png` with your new logo
- [ ] Replace `/public/favicon.ico` with your new favicon ([Free PNG to ICO converter](https://www.freeconvert.com/png-to-ico))
