Create a dynamic header featuring your logo on the left,
links in the center, and a Call to Action (CTA) on the right. On mobile devices,
the links and CTA are concealed and can be accessed through a burger menu.

<Frame>
    <img src="/images/components/header.avif" />
</Frame>

Insert your own logo into the `/public/images` folder, ensuring that the file name in the component matches (currently set as logo.svg).

```html resources/js/Components/Header.vue
<div class="max-w-7xl m-auto navbar bg-base-100">
    <div class="navbar-start">
        <div class="dropdown">
            <div tabindex="0" role="button" class="btn btn-ghost lg:hidden">

            </div>
            <ul tabindex="0" class="menu menu-sm dropdown-content mt-3 z-[1] p-2 shadow bg-base-100 rounded-box w-52">
                <li><a href="#">About Us</a></li>
                <li><a href="#pricing">Pricing</a></li>
                <li><a href="#">How It Works</a></li>

    </div>
</div>
```
<Tip>
For more variants check [https://daisyui.com/components/navbar/](https://daisyui.com/components/navbar/)
</Tip>
