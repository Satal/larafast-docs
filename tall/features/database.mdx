By default, LaraFast uses an SQLite database. Feel free to change it to MySQL or PostgreSQL in .env configs

```dotenv .env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravelstarterkit
DB_USERNAME=root
DB_PASSWORD=
```
